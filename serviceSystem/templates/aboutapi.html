API 설명

다수의 지역에 대한 행정동 값을 입력 받아 해당 동에 부합하는 공시지가와 인구밀도에 대한 점수를 계산해 순위를 산출하는 API입니다.

GET 방식으로 요청하면 응답은 JSON 형식으로 지원됩니다.

다양한 업종의 카테고리를 연계하여 업종의 분포도를 보는데 활용 가능합니다.
EX) 방배동 + 서점 , 서초동 + 서점
